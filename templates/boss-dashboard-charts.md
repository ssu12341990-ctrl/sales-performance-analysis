# 老板看板图表方案 v1.0

## 1. 看板定位
本看板面向老板/管理层，核心目标：
- **一眼看经营结果**（KPI）
- **一眼看趋势变化**（折线图）
- **一眼看团队对比**（柱状图）
- **一眼看风险预警**（风险榜）

---

## 2. 页面布局建议

```
┌─────────────────────────────────────────────┐
│  顶部 KPI 卡区（6个核心指标）                    │
├───────────────────┬─────────────────────────┤
│  近30天业绩趋势图   │  近30天净业绩趋势图        │
├───────────────────┼─────────────────────────┤
│  三公司业绩对比图   │  销售TOP10排行图           │
├───────────────────┴─────────────────────────┤
│  付款结构占比图（环形图）                        │
├─────────────────────────────────────────────┤
│  风险区：退款率TOP10 | 尾款未收TOP10 | 无开单名单 │
└─────────────────────────────────────────────┘
```

---

## 3. 顶部 KPI 卡（6个）

| 卡片名称 | 指标口径 | 显示格式 | 颜色建议 |
|---|---|---|---|
| 本月总业绩 | SUM(实收业绩) | ¥金额 | 蓝色 |
| 本月净业绩 | SUM(净业绩) | ¥金额 | 绿色 |
| 今日业绩 | 当日实收合计 | ¥金额 | 蓝色 |
| 本周业绩 | 本周实收合计 | ¥金额 | 蓝色 |
| 月目标完成率 | 本月总业绩/月目标 | 百分比+进度条 | 绿色/橙色/红色 |
| 退款率 | ABS(退款)/实收业绩 | 百分比 | 低<5%绿，5-10%橙，>10%红 |

---

## 4. 中部图表（5个）

### 图表1：近30天业绩趋势图
| 属性 | 说明 |
|---|---|
| 图表类型 | 折线图 |
| 横轴 | 日期（近30天）|
| 纵轴 | 实收业绩（元）|
| 数据来源 | Sheet1 业绩流水表，按日汇总 |
| 重点标注 | 标注最高点/最低点 |
| 用途 | 看每日业绩波动，识别高峰低谷 |

---

### 图表2：近30天净业绩趋势图
| 属性 | 说明 |
|---|---|
| 图表类型 | 折线图 |
| 横轴 | 日期（近30天）|
| 纵轴 | 净业绩（元）|
| 数据来源 | Sheet1，按日汇总实收+退款 |
| 重点标注 | 退款发生当日会有明显下降 |
| 用途 | 看真实净业绩波动，识别退款冲击 |

---

### 图表3：三公司业绩对比图
| 属性 | 说明 |
|---|---|
| 图表类型 | 簇状柱形图 |
| 横轴 | 销售公司名称 |
| 纵轴 | 实收业绩（元）|
| 数据来源 | Sheet4 公司业绩汇总表 |
| 可追加 | 同时显示净业绩对比（双柱）|
| 用途 | 老板快速对比三个公司谁贡献最大 |

---

### 图表4：销售个人TOP10排行图
| 属性 | 说明 |
|---|---|
| 图表类型 | 水平条形图 |
| 横轴 | 净业绩（元）|
| 纵轴 | 销售姓名 |
| 数据来源 | Sheet3 销售业绩汇总表，取TOP10 |
| 排序 | 净业绩降序 |
| 用途 | 老板快速看谁是销售明星 |

---

### 图表5：付款结构占比图
| 属性 | 说明 |
|---|---|
| 图表类型 | 环形图 |
| 分类 | 全款 / 报名费 / 尾款 |
| 数值 | 各类型金额占总实收业绩比例 |
| 数据来源 | Sheet4 公司业绩汇总表 |
| 用途 | 看付款结构健康度，全款占比越高越好 |

---

## 5. 底部风险区（3个榜单）

### 风险榜1：退款率TOP10销售
| 字段 | 说明 |
|---|---|
| 销售姓名 | 销售人员 |
| 销售公司 | 所属公司 |
| 实收业绩 | 总实收 |
| 退款金额 | 退款总额 |
| 退款率 | ABS(退款)/实收，降序排列 |

**预警规则：**
- 退款率 > 10% → 红色标注
- 退款率 5%-10% → 橙色标注
- 退款率 < 5% → 正常

---

### 风险榜2：尾款未收TOP10
| 字段 | 说明 |
|---|---|
| 客户姓名 | 客户 |
| 销售姓名 | 责任销售 |
| 销售公司 | 所属公司 |
| 应收尾款 | 订单应收尾款 |
| 已收尾款 | 已收金额 |
| 未收尾款 | 应收-已收，降序排列 |
| 超期天数 | 距首次付款天数 |

**预警规则：**
- 超期 > 30天 → 红色标注
- 超期 15-30天 → 橙色标注
- 超期 < 15天 → 正常

---

### 风险榜3：连续无开单销售名单
| 字段 | 说明 |
|---|---|
| 销售姓名 | 销售人员 |
| 销售公司 | 所属公司 |
| 销售团队 | 所属团队 |
| 最近开单日期 | 最近一次成交日期 |
| 连续无开单天数 | 当前日期-最近开单日期 |

**预警规则：**
- 连续无开单 > 7天 → 橙色标注
- 连续无开单 > 14天 → 红色标注

---

## 6. 图表颜色规范

| 用途 | 建议颜色 |
|---|---|
| 正向指标（业绩/完成率高）| 蓝色 #1890FF / 绿色 #52C41A |
| 负向指标（退款/风险）| 红色 #FF4D4F |
| 中间状态（待追踪）| 橙色 #FA8C16 |
| 背景 | 深色 #001529 或浅色 #F5F5F5 |
| KPI卡片背景 | 白色，边框阴影 |
