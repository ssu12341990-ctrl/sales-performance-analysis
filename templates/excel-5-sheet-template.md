## Sheet1：业绩流水表

### 用途
记录每一笔业务动作：报名费 / 尾款 / 全款 / 退款
适合做：日业绩、周/月业绩、去退款业绩、某日进线带来的业绩

### 表头
```
业务日期,公司主体,销售公司,销售团队,销售姓名,客户姓名,客户电话,客户进线日期,客户来源,产品名称,订单编号,类型,金额,备注
```

### 字段说明
| 字段 | 说明 |
|---|---|
| 业务日期 | 实际收款/退款发生日期 |
| 公司主体 | 归属公司主体 |
| 销售公司 | 归属销售公司 |
| 销售团队 | 归属团队 |
| 销售姓名 | 成交销售 |
| 客户姓名 | 客户名称 |
| 客户电话 | 客户手机号 |
| 客户进线日期 | 客户首次进线日期 |
| 客户来源 | 抖音/快手/转介绍等 |
| 产品名称 | 成交产品/课程 |
| 订单编号 | 唯一订单编号 |
| 类型 | 报名费/尾款/全款/退款 |
| 金额 | 实收金额，退款填负数 |
| 备注 | 特殊说明 |

## Sheet2：客户订单汇总表

### 用途
一行代表一个客户/订单的汇总结果。
适合老板直接查看全款客户、报名费客户、未补尾款客户、已退款客户。

### 表头
```
开单日期,公司主体,销售公司,销售团队,销售姓名,客户姓名,客户电话,客户进线日期,客户来源,产品名称,订单编号,报名费,尾款,全款,退款,实收业绩,净业绩,尾款状态,付款类型,退款状态,退款日期,备注
```

### 关键公式（假设列位置）
| 公式 | 说明 |
|---|---|
| =SUM(L2:N2) | 实收业绩 = 报名费+尾款+全款 |
| =P2+O2 | 净业绩 = 实收业绩+退款 |

### 尾款状态取值
- 无尾款（全款客户）
- 未补尾款
- 已补尾款

### 付款类型取值
- 全款
- 报名费
- 报名费+尾款

### 退款状态取值
- 未退款
- 已退款
- 部分退款

## Sheet3：销售业绩汇总表

### 用途
按销售汇总周期业绩，适合个人业绩排名、尾款回收率对比。

### 表头
```
统计周期,销售公司,销售团队,销售姓名,成交客户数,报名费客户数,全款客户数,已补尾款客户数,未补尾款客户数,实收业绩,净业绩,退款金额,尾款金额,尾款回收率,客单价,个人排名
```

### 关键公式
| 公式 | 说明 |
|---|---|
| =IF(E2=0,0,J2/E2) | 客单价 = 实收业绩/成交客户数 |
| =RANK(K2,$K$2:$K$100,0) | 个人排名（按净业绩降序）|
| =IF((H2+I2)=0,0,H2/(H2+I2)) | 尾款回收率 |

## Sheet4：公司业绩汇总表

### 用途
按公司主体/销售公司汇总，适合三公司对比、目标完成率追踪。

### 表头
```
统计周期,公司主体,销售公司,成交客户数,实收业绩,净业绩,全款金额,报名费金额,尾款金额,退款金额,退款率,尾款未收金额,尾款回收率,目标金额,目标完成率
```

### 关键公式
| 公式 | 说明 |
|---|---|
| =IF(E2=0,0,ABS(J2)/E2) | 退款率 = ABS(退款)/实收业绩 |
| =IF(N2=0,0,E2/N2) | 目标完成率 = 实收业绩/目标金额 |
| =IF((I2+L2)=0,0,I2/(I2+L2)) | 尾款回收率 |

## Sheet5：老板看板

### 用途
给老板看的经营总览页，放 KPI、图表、风险榜。

### 顶部 KPI 区
| 指标 | 公式/口径 |
|---|---|
| 本月总业绩 | SUM(Sheet1 当月金额) |
| 本月净业绩 | 总业绩 + 退款金额 |
| 今日业绩 | SUM(今日实收) |
| 本周业绩 | SUM(本周实收) |
| 月目标完成率 | 本月总业绩 / 月目标 |
| 退款率 | ABS(退款金额) / 实收业绩 |

### 中部图表区
| 图表名称 | 类型 | 数据来源 |
|---|---|---|
| 近30天业绩趋势 | 折线图 | Sheet1 按日汇总 |
| 近30天净业绩趋势 | 折线图 | Sheet1 按日汇总 |
| 三公司业绩对比 | 簇状柱形图 | Sheet4 |
| 销售TOP10 | 条形图 | Sheet3 |
| 付款结构占比 | 环形图 | Sheet4 全款/报名费/尾款 |

### 底部风险区
| 榜单 | 字段 | 数据来源 |
|---|---|---|
| 退款率TOP10销售 | 销售姓名,净业绩,退款金额,退款率 | Sheet3 |
| 尾款未收TOP10 | 客户姓名,销售姓名,未收尾款,超期天数 | Sheet2 |
| 连续无开单销售 | 销售姓名,最近开单日期,连续无开单天数 | Sheet3 |
