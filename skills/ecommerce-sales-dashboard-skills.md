# 电商业绩分析老板看板 Skills v1.0

## 1. Skills 定位
本 Skills 用于持续沉淀电商销售业绩分析的规范、模板和口径。
后续每次新增需求，直接追加到本文件末尾，不推翻已有内容。

---

## 2. 适用场景
- 每天每个公司主体总业绩 / 三个销售公司总业绩
- 某日进线数据业绩查询
- 某销售业绩查询
- 全款客户查询（可查询追尾款率高的销售）
- 报名费客户查询（无尾款）（可查询追尾款率低的销售）
- 周/月销售公司业绩 / 个人业绩
- 周/月销售公司业绩 / 个人业绩（去退款业绩）

---

## 3. 核心表结构

### 3.1 业绩流水表（Sheet1）
> 一行 = 一笔业务动作

**表头：**
```
业务日期,公司主体,销售公司,销售团队,销售姓名,客户姓名,客户电话,客户进线日期,客户来源,产品名称,订单编号,类型,金额,备注
```
- 类型：报名费 / 尾款 / 全款 / 退款
- 金额：退款填负数

---

### 3.2 客户订单汇总表（Sheet2）
> 一行 = 一个客户/订单汇总

**表头：**
```
开单日期,公司主体,销售公司,销售团队,销售姓名,客户姓名,客户电话,客户进线日期,客户来源,产品名称,订单编号,报名费,尾款,全款,退款,实收业绩,净业绩,尾款状态,付款类型,退款状态,退款日期,备注
```

**公式：**
- 实收业绩 = 报名费 + 尾款 + 全款
- 净业绩 = 实收业绩 + 退款

---

### 3.3 销售业绩汇总表（Sheet3）
> 一行 = 一个销售的周期汇总

**表头：**
```
统计周期,销售公司,销售团队,销售姓名,成交客户数,报名费客户数,全款客户数,已补尾款客户数,未补尾款客户数,实收业绩,净业绩,退款金额,尾款金额,尾款回收率,客单价,个人排名
```

---

### 3.4 公司业绩汇总表（Sheet4）
> 一行 = 一个公司主体/销售公司的周期汇总

**表头：**
```
统计周期,公司主体,销售公司,成交客户数,实收业绩,净业绩,全款金额,报名费金额,尾款金额,退款金额,退款率,尾款未收金额,尾款回收率,目标金额,目标完成率
```

---

### 3.5 老板看板（Sheet5）
**KPI 卡：** 本月总业绩 / 本月净业绩 / 今日业绩 / 本周业绩 / 月目标完成率 / 退款率

**图表：**
- 近30天业绩趋势折线图
- 三公司业绩对比柱状图
- 销售TOP10条形图
- 付款结构环形图

**风险榜：**
- 退款率TOP10销售
- 尾款未收TOP10
- 连续无开单销售名单

## 4. 业务口径规范

### 4.1 业绩口径
| 指标 | 口径 |
|---|---|
| 实收业绩 | 报名费 + 尾款 + 全款 |
| 净业绩 | 实收业绩 + 退款（退款为负数）|
| 总业绩 | 所有实收业绩汇总 |
| 去退款业绩 | 所有净业绩汇总 |

### 4.2 客户分类口径
| 客户类型 | 判断条件 |
|---|---|
| 全款客户 | 全款金额 > 0 |
| 报名费客户 | 报名费 > 0 且 全款 = 0 |
| 已补尾款客户 | 尾款 > 0 |
| 未补尾款客户 | 报名费 > 0 且 尾款 = 0 且 全款 = 0 |
| 退款客户 | 退款 <> 0 |

### 4.3 尾款口径
| 指标 | 公式 |
|---|---|
| 尾款回收率 | 已收尾款 / 应收尾款 |
| 尾款未收金额 | 应收尾款 - 已收尾款 |

### 4.4 退款口径
| 指标 | 公式 |
|---|---|
| 退款金额 | 退款字段汇总（负数）|
| 退款率 | ABS(退款金额) / 实收业绩 |
